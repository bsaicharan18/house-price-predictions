🏠 House Price Prediction Website

📌 Overview

This project is a web-based application that predicts house prices using machine learning.
Users can enter house details and get an estimated price instantly.

---

🚀 Features

- Predicts house prices based on input data
- Simple and user-friendly web interface
- Machine learning model integration
- Fast predictions

---

🛠️ Technologies Used

- Python
- Flask
- HTML (Frontend)
- Scikit-learn
- Pandas & NumPy

---

📂 Project Structure

house-price-prediction/

│
├── app.py # Main Flask application
├── model.pkl # Trained ML model
├── house_price_practice.csv # Dataset
│
├── templates/
│ └── index.html # Frontend page
│
└── README.md

---

▶️ How to Run the Project

1. Install required libraries:
   pip install flask pandas scikit-learn

2. Run the application:
   python app.py

3. Open your browser and go to:
   http://127.0.0.1:5000/

---

📊 Model Details

- Algorithm Used: Regression Model
- The model is trained using house price dataset
